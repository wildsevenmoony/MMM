# **Moony's Magnificent Modules** - Update 27.06.2026
-# [Mod Page](https://steamcommunity.com/sharedfiles/filedetails/?id=3455903038)

## Added
- ACE Arsenal module for Eden and Zeus
- Optional scroll-wheel action for ACE Arsenal objects
- Arsenal blacklist and limited-whitelist modes
- Optional compatible magazine and attachment expansion for limited arsenals
- Set Respawn Timer selector for sides, groups and players when the Zeus module is not placed on a unit
- Option to apply respawn timer changes once or for the whole mission
- Module to add MMA Zeus self-action eligibility when MMA is loaded

- Nothing

- Mobile HQ Zeus module can update an existing MHQ on the target instead of creating a duplicate
- Mobile HQ modules can choose whether the MHQ starts deployed

- SwitchMove and PlayMove animation handling was improved for multiplayer
- Respawn timer defaults and limits were refined for live Zeus use
